# Falsy Remover

A small JavaScript utility that returns a new array containing only truthy values. (fCC Lab).
The function filters out all falsy elements (`false`, `null`, `0`, `""`, `undefined`, `NaN`) without modifying the original input array.

## Features
- Removes all falsy values from an array  
- Preserves the original array (no mutation)  
- Returns a clean array containing only truthy elements  
- Simple and readable implementation
